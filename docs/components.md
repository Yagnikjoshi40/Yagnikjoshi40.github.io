# Component Documentation

## Hero
- `HeroSection.tsx` assembles the six-layer hero treatment: particles, floating gems, premium copy, CTAs, trust badges, and scroll indicator.
- `MorphingText.tsx` rotates between “ELEGANCE” and “LUXURY” every five seconds.
- `ParticleBackground.tsx` renders the floating luxury particle field.

## Products
- `ProductGrid.tsx` renders responsive collection cards.
- `ProductCard.tsx` includes hover lift, urgency badge, floating actions, and 3D-inspired pricing.
- `ProductFilters.tsx` and `SearchBar.tsx` provide premium discovery controls.

## Product Detail
- `ProductDetailPage.tsx` composes the 3-column PDP experience.
- `ThreeDViewer.tsx` provides a React Three Fiber viewer with studio lighting and orbit controls.
- `ReviewCarousel.tsx` and `RelatedProducts.tsx` handle social proof and upsell blocks.

## Commerce
- `CartPage.tsx` and checkout components deliver a luxury shopping funnel with sticky summary and confirmation state.

## Common
- `Navbar.tsx`, `Footer.tsx`, `CustomCursor.tsx`, `AnimatedButton.tsx`, and `WhatsAppWidget.tsx` define shared brand interactions.
